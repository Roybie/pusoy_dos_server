# Pusoy Dos

A multiplayer Pusoy Dos server written in [Rust](https://www.rust-lang.org).

## Prerequisites
- Docker

## Build
- Checkout source
- `make docker-build` 
- `make docker-run`

## Tech
- [rust](https://www.rust-lang.org)
 - [iron framework](http://ironframework.io/)
 - [tera templates](https://github.com/Keats/tera)
- mysql
- docker
- elm?
